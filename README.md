# English Learning App

C# desktop learning app for elementary grades — quizzes with audio and writing practice

Maintainer: Zhassulan Baimyshev — https://www.linkedin.com/in/zhassulan-baimyshev/

Summary

A desktop educational application that presents tests for learning English vocabulary, including audio playback and written exercises. Uses a local database for progress tracking.

How to run

1. Clone and open `english.sln` in Visual Studio.
2. Restore packages, build and run.

Notes

- Useful as a learning prototype; consider extracting media assets into a `resources/` folder for clarity.
